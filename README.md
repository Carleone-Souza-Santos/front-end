<h1>Blog</h1>

<p>Projeto feito com o intuito de aprimorar o aprendizado com javascript.</p>

<p>Utilizando o Materialize CSS, que é um framework Material Design (diretrizes do Google), para estilizar os componentes, como botões, cards e inputs.</p>

<h3>Funções e conceitos importantes:</h3>
<p>Manipulação do DOM (Document Object Model) para alterar o conteúdo da página de forma dinâmica.
Função renderizarNoticias: Atualiza a lista de notícias no HTML.
Função adicionarNoticias: Adiciona uma nova notícia ao banco de dados (array).
Função removerNoticias: Remove uma notícia do banco de dados baseado no seu id.
Uso de Date.now() para gerar um identificador único para cada notícia.
Manipulação de arrays e objetos para armazenar e gerenciar os dados das notícias.</p>

<h3>Conceitos de Front-End:</h3>
<p>Manipulação de Formulários: O projeto possui campos de entrada para nome, descrição e imagem da notícia, que são manipulados com JavaScript.
Eventos de Clique: Ao clicar no botão de adicionar notícia, a função correspondente é executada, e o conteúdo é inserido no banco de dados.
Interatividade com o Usuário: Exibição e remoção de notícias com base nas ações do usuário</p>

<p>Esse projeto é uma ótima base para aprender conceitos essenciais de desenvolvimento web, como a manipulação do DOM e a interatividade com o usuário.</p>
